# Julio Palacio - Terminal Portfolio

My personal website built as an interactive terminal. Type commands to learn about me, my work, and how to get in touch.

**Live:** [palacioj.com](https://palacioj.com)

## About

A terminal-styled portfolio where visitors navigate using command-line commands like `about`, `resume`, `sumfetch`, and more. Built with Next.js and inspired by [LiveTerm](https://github.com/Cveinnt/LiveTerm).

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Setup

```bash
yarn install
yarn dev
```

Open [localhost:3000](http://localhost:3000).

## Deployment

Pushes to `main` automatically deploy to the server via GitHub Actions.
