## üöÄ Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS version recommended)
- [Yarn](https://yarnpkg.com/) package manager

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/jpalacio0x/Palacioj.git
    cd Palacioj
    ```

2.  **Install dependencies:**
    ```bash
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    yarn dev
    ```

4.  **View the application:**
    Open [http://localhost:3000](http://localhost:3000) in your browser.

## üìù Usage

Once the terminal is loaded, type `help` to view a list of all available commands. 

Example commands:
- `whoami`: Displays the current user.
- `repo`: Opens this GitHub repository.
- `email`: Opens your default mail client to contact me.
